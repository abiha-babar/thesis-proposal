\section{Project Plan}
\label{sec:plan}

The table below shows the step-by-step plan for completing this thesis. Reading, writing, and practical work happen at the same time throughout the project.

\begin{table}[h!]
    \centering
    \begin{tabularx}{\textwidth}{ | c | X | X | }
        \hline
        \textbf{Time} & \textbf{Reading / Writing} & \textbf{Practical Work} \\
        \hline
        Weeks 1--3   & Read all related papers on tool use, agent evaluation, and context saturation; finish the literature review & Install and test Ollama with \texttt{gemma4:4b} and \texttt{qwen3.5:4b}; set up API access for Gemini 3.1 Pro and GPT-4o; install and verify OpenClaw \\
        \hline
        Weeks 4--5   & Write up the literature review chapter & Design the 20 skill files; write the \texttt{SKILL.md} template; draft applicability conditions, execution policies, and termination criteria for all three skill domains \\
        \hline
        Weeks 6--7   & Write the thesis introduction and the three research questions & Implement all 20 \texttt{SKILL.md} files and their equivalent JSON schema versions for Condition~A; test both setups end-to-end on a few sample tasks \\
        \hline
        Weeks 8--9   & Write the methodology chapter & Build the full set of 40 tasks across three difficulty levels; prepare answer keys and ground-truth policy graphs for scoring; write the Python scripts that read OpenClaw's JSONL logs and compute metrics \\
        \hline
        Weeks 10--12 & Document the experiment setup and the measurement tools used & Run all 960 agent runs (40 tasks $\times$ 2 conditions $\times$ 4 models $\times$ 3 repeats); save all logs, token counts, latency readings, and energy measurements \\
        \hline
        Weeks 13--15 & Write the results chapter & Process all log files; calculate task completion rates, hallucination counts, trajectory compliance scores, token usage, latency, and cost; produce all charts and tables \\
        \hline
        Weeks 16--17 & Write the discussion and conclusion chapters & Run sensitivity checks to confirm results are stable across different task orderings \\
        \hline
        Weeks 18--19 & Send the first complete draft to the supervisor for feedback & Finalise all appendices: skill files, task list, Python scripts, raw data tables \\
        \hline
        Weeks 20--22 & Make all corrections based on feedback & Submit the final completed thesis \\
        \hline
    \end{tabularx}
    \caption{Planned Project Timeline}
    \label{tab:timeline}
\end{table}
